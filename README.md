# weatherapp


The weather app will be a simple yet dynamic web application that allows users to check current weather conditions
for any location. It will use HTML for structuring the layout, CSS for styling the user interface, and JavaScript for functionality. 
The core feature of the app is its integration with the WeatherAPI service, which provides real-time weather data.


Here’s an overview of how it will work:

Input Field: Users will enter a city or location name in a search field.

API Call: When the user submits their search, JavaScript sends a request to the WeatherAPI, using the location entered.

Data Display: Once the data is retrieved, it will be displayed on the page, showing key information like temperature, weather conditions, humidity, and wind speed.

Responsive Design: The app will be styled with CSS for a clean, visually appealing look, optimized for both desktop and mobile screens.

This project will highlight how to call an external API, handle responses, and dynamically update the web page using JavaScript.
